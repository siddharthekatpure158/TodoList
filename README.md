📝 To-Do List Application (Core Java)
📌 Overview

The To-Do List Application is a simple console-based project developed using Core Java.
It uses the ArrayList class from the Java Collections Framework to dynamically manage tasks in memory.

This project demonstrates basic Java programming concepts such as:

OOP principles

Collections Framework

User input handling

Looping and conditional statements

🚀 Features

➕ Add new task

📋 View all tasks

✏️ Update existing task

❌ Delete task

📌 Mark task as completed (if implemented)

🛠️ Technologies Used

Core Java

ArrayList (Java Collections Framework)

Scanner Class for user input

Eclipse / IntelliJ IDE
